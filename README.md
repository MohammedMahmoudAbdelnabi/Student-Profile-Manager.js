# JavaScript Object Reference Demo 🧠

A simple JavaScript project that demonstrates how **objects behave in memory** and how **object references work**.

The project shows how updating an object property affects all variables that reference the same object. It also demonstrates modifying object properties and using template strings to display data.

This project is part of my JavaScript learning journey to better understand how JavaScript handles objects and references.

---

## 📌 Features

* Create a JavaScript object
* Access object properties
* Update object properties
* Use template strings to print object data
* Demonstrate **object reference behavior**
* Show how modifying one reference affects the original object

---

## 🧠 Concepts Used

This project practices important JavaScript fundamentals:

* JavaScript Objects
* Object Properties
* Updating Object Values
* Object References
* Template Literals
* Variables
* Console Output
